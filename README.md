#Usage
Initiate and perform installation using:
```yarn install```
Start your server with:
`yarn dev`

The following are the list of the available endpoint you might want to query or build up on:
1. POST /me
2. GET /shipments/
3. GET /shipments/:id
4. GET /autosuggest-airports?type=origin&q=FRA
5. GET /searches/065a8640-effa-4b2e-a971-5d7b71265f77
6. GET /product-config
7. GET /search-results
8. GET /get-dynamic-price-indicator
9. GET /offers/bb999b11-fba5-4dc0-96c3-326c640f7a2b
10. GET /get-entity-metadata
